# 10 Responsive Sites

A collection of ten premium, fully responsive websites — each with a unique aesthetic theme, built with pure HTML, CSS, and JavaScript. No frameworks. No UI libraries. Just clean, semantic, hand-crafted code.

## Tech Stack

- HTML5 — semantic, accessible markup
- CSS3 — custom properties, grid, flexbox, animations, backdrop-filter
- JavaScript — vanilla, no dependencies
- Google Fonts — unique typeface per project

## Features

- Every site is fully responsive across mobile, tablet, and desktop
- Each project has a distinct color palette and typographic personality
- Modern CSS techniques: CSS Grid, Flexbox, custom properties, scroll-snap, clip-path, backdrop-filter
- Micro-animations and hover interactions throughout
- Semantic HTML with unique IDs on all interactive elements
- No comments in production code

## Design Highlights

**Flavour** uses a `#111` dark base with `#f59e0b` amber accents, glowing food cards, and Font Awesome icons in an animated hero grid.

**Kreative** uses a cream editorial background with electric `#8b5cf6` and `#ec4899` accents, bold sans-serif headlines.

**CyberFolio** uses Orbitron monospace font, `#00ffe1` neon teal glows, CSS scanlines, and animated skill bars.

**Prestige Motors** uses Special Elite serif, deep `#0c0b09` carbon with `#c9a84c` gold accents, diamond dividers, and hover border animations.

**Archaus** uses Cormorant Garamond with a split-panel hero, overlay slide-in nav, floating form labels on contact, and a dark team section.

**The Road** uses a deep ocean `#0a1628` base with `#0891b2` teal, SVG wave animations, and flip-card tour pricing.

**Vino House** uses `#7c1a24` burgundy with `#c46b7a` rose, a full-screen slideshow with dot navigation and animated floating bottle.

**Apex Store** uses a pure white Apple-inspired layout with scroll-snap full-page sections and floating device animations.

**NexAgency** uses `#0f1c2e` navy with `#059669` emerald, animated background blobs, rotating headline words, and pricing hover-reveal.

**PayVault** uses a two-column layout with a sticky auth sidebar, indigo/teal gradients, mega dropdown navigation, and multi-step signup flow.
